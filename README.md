# Sheffield AI Blog

## Deployment Steps

1. **Install Dependencies**: Ensure all packages are installed.
   ```bash
   pip install -r requirements.txt
   ```

2. **Run Locally**: Start a local server to preview your site.
   ```bash
   mkdocs serve -w .
   ```

3. **Deploy to GitHub Pages**: Go live with:
   ```bash
   mkdocs gh-deploy
   ```l
